# Security Policy

Security fixes target the latest release and the `main` branch.

Report undisclosed vulnerabilities through GitHub's **Report a vulnerability** flow in the repository Security tab, not a public issue. Include affected versions, reproduction steps, impact, and mitigation ideas. An acknowledgement is expected within 7 days, followed by coordinated disclosure for confirmed issues.

Do not include real payment instructions, account numbers, credentials, sanctions data, or personal information. BankBridge is an educational sandbox built for synthetic data; it is not certified for production banking, settlement, AML, or sanctions decisions.
